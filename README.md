# dashboard_frontend
Front end code for the dashboard.

# Installation 

1) Enter the directory
`cd dash_heroku/`

2) Make a virtulenv env

`python3 -m venv env`

3) Activate the env

`python3 -m venv env`

4) Install requirements 

`pip install -r requirements.txt`

5) Start the app 
`python index.py`
